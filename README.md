# ⇄ Smart Layout Switcher (Double Ctrl+C)

Легкий и надежный Python-скрипт для мгновенного перевода выделенного текста между русской и английской раскладками клавиатуры. 

Скрипт разработан специально для обхода жестких политик безопасности **Windows 10/11 Enterprise (Корпоративная)**. Он **не использует** опасные низкоуровневые хуки клавиатуры (которые часто блокируются антивирусами, AppLocker или GPO как кейлоггеры) и работает на чистом Win32 API.

## 🚀 Как это работает

1. Выделите текст, набранный не в той раскладке (например: `ghbdtn` или `Yt hf,jnftn&`).
2. Быстро нажмите комбинацию **Ctrl + C + C** (двойное копирование, как дабл-клик мыши).
3. Скрипт мгновенно переведет текст внутри буфера обмена.
4. Нажмите **Ctrl + V** — готово! Текст заменен на `привет` или `Не работает?`.

## ✨ Особенности

- **Полная поддержка спецсимволов**: Корректно переводит цифровой ряд (`!`, `@`, `#`, `$`, `%`, `^`, `&`, `*`, `(`, `)`) и знаки препинания (`?`, `:`, `;`, `.` , `,`) в соответствии с раскладками Windows.
- **Работает в Windows Enterprise**: Не требует прав администратора (UAC) для запуска.
- **Иконка в системном трее**: Удобное меню управления возле часов. Можно поставить мониторинг на паузу или полностью закрыть приложение в один клик.
- **Низкое потребление ресурсов**: Потребляет минимум оперативной памяти и не нагружает процессор.

## 🛠 Установка и запуск из исходников

### Требования
У вас должен быть установлен Python 3.8 или выше.

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   cd НАЗВАНИЕ_REПОЗИТОРИЯ
   ```

2. Установите зависимости:
   ```bash
   pip install pyperclip pystray pillow
   ```

3. Запустите скрипт:
   ```bash
   python main.py
   ```

## 📦 Сборка автономного .EXE файла

Вы можете собрать скрипт в один исполняемый файл, чтобы переносить его на флешке или запускать на компьютерах без установленного Python.

1. Установите PyInstaller:
   ```bash
   pip install pyinstaller
   ```

2. Скомпилируйте проект в фоновый режим (флаг `-w` уберет появление черного окна консоли):
   ```bash
   pyinstaller -F -w main.py
   ```

3. Заберите готовый файл `main.exe` из папки **`dist/`**.

## 🔄 Автозагрузка в Windows

Чтобы скрипт автоматически запускался при старте операционной системы:
1. Нажмите комбинацию клавиш **Win + R**.
2. Введите команду `shell:startup` и нажмите Enter (откроется папка Автозагрузки).
3. Перетащите в эту папку ярлык вашего скомпилированного файла `main.exe`.
