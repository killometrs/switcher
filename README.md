## 📖 Описание

**Smart Layout Switcher** — это утилита для мгновенного перевода выделенного текста между русской и английской раскладками клавиатуры.

Скрипт разработан специально для обхода жестких политик безопасности **Windows 10/11 Enterprise (Корпоративная)**. Он **не использует** опасные низкоуровневые хуки клавиатуры (которые часто блокируются антивирусами, AppLocker или GPO как кейлоггеры) и работает на чистом Win32 API.

---

## 🚀 Как это работает

1. **Выделите текст**, набранный не в той раскладке (например: `ghbdtn` или `Yt hf,jnftn&`).
2. Быстро нажмите комбинацию **Ctrl + C + C** (двойное копирование, как дабл-клик мыши).
3. Скрипт мгновенно переведет текст внутри буфера обмена.
4. Нажмите **Ctrl + V** — готово! Текст заменен на `привет` или `Не работает?`.

## ✨ Особенности

- ✅ **Полная поддержка спецсимволов**: Корректно переводит цифровой ряд (`!`, `@`, `#`, `$`, `%`, `^`, `&`, `*`, `(`, `)`) и знаки препинания (`?`, `:`, `;`, `.`, `,`) в соответствии с раскладками Windows.
- ✅ **Работает в Windows Enterprise**: Не требует прав администратора (UAC) для запуска.
- ✅ **Иконка в системном трее**: Удобное меню управления возле часов. Можно поставить мониторинг на паузу или полностью закрыть приложение в один клик.
- ✅ **Низкое потребление ресурсов**: Потребляет минимум оперативной памяти и не нагружает процессор.
- ✅ **Безопасность**: Не использует низкоуровневые хуки, не является кейлоггером.
- ✅ **Портативность**: Работает без установки, можно запускать с флешки.

## 🛠 Установка и запуск из исходников

### Способ 1: Установка из исходников

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/killometrs/switcher.git
cd switcher
```

2. **Установите зависимости**
```bash
pip install -r requirements.txt
```

Или вручную:
```bash
pip install pyperclip pystray pillow
```

3. **Запустите скрипт**
```bash
python switcher.py
```

### Способ 2: Создание requirements.txt

Создайте файл `requirements.txt` со следующим содержимым:
```txt
pyperclip>=1.8.2
pystray>=0.19.0
pillow>=10.0.0
```

## 📦 Сборка автономного .EXE файла

Вы можете собрать скрипт в один исполняемый файл, чтобы переносить его на флешке или запускать на компьютерах без установленного Python.

### Шаг 1: Установка PyInstaller
```bash
pip install pyinstaller
```

### Шаг 2: Сборка EXE

#### Без окна консоли (рекомендуется для фоновой работы)
```bash
pyinstaller -F -w -n "LayoutSwitcher" switcher.py
```

#### С окном консоли (для отладки)
```bash
pyinstaller -F -n "LayoutSwitcher" switcher.py
```

#### С иконкой (если есть .ico файл)
```bash
pyinstaller -F -w -n "LayoutSwitcher" --icon=icon.ico switcher.py
```

### Шаг 3: Готовый файл
Заберите готовый файл `LayoutSwitcher.exe` из папки **`dist/`**.

### Дополнительные опции PyInstaller

| Флаг | Описание |
|------|----------|
| `-F` | Собрать в один файл |
| `-w` | Убрать окно консоли (для фоновых приложений) |
| `-n "Name"` | Задать имя выходного файла |
| `--icon=icon.ico` | Добавить иконку к EXE |
| `--onefile` | Собрать в один файл (альтернатива -F) |

## 🏢 Использование в Windows Enterprise

### Особенности работы в корпоративной среде

- ✅ **Не требует прав администратора** — запускается в пользовательском режиме
- ✅ **Не использует низкоуровневые хуки** — не блокируется антивирусами
- ✅ **Не вызывает UAC** — не требует подтверждения при запуске
- ✅ **Работает через Win32 API** — использует стандартные системные вызовы
- ✅ **Не является кейлоггером** — не перехватывает нажатия клавиш

### Почему это безопасно для Enterprise?

1. **Нет системных хуков** — не перехватывает события клавиатуры глобально
2. **Только буфер обмена** — работает исключительно с текстом
3. **Win32 API** — использует стандартные и безопасные системные вызовы
4. **Прозрачный код** — весь код открыт и доступен для проверки

### Проверка безопасности

Для проверки безопасности в вашей организации:
1. Исходный код доступен в репозитории
2. Можно провести аудит кода
3. Нет скрытых сетевых запросов
4. Нет доступа к файловой системе (кроме самого скрипта)
5. Нет записи в реестр (кроме автозагрузки, если настроена)

## 🔄 Автозагрузка в Windows

Чтобы скрипт автоматически запускался при старте операционной системы:

### Способ 1: Через папку автозагрузки (рекомендуется)

1. Нажмите комбинацию клавиш **Win + R**
2. Введите команду `shell:startup` и нажмите Enter
3. Откроется папка Автозагрузки
4. Перетащите в эту папку ярлык вашего скомпилированного файла `LayoutSwitcher.exe`

### Способ 2: Через планировщик задач

1. Откройте **Планировщик задач** (taskschd.msc)
2. Нажмите **"Создать задачу"**
3. На вкладке **"Триггеры"** выберите **"При входе в систему"**
4. На вкладке **"Действия"** укажите путь к `LayoutSwitcher.exe`
5. На вкладке **"Параметры"** отметьте **"Запускать с наивысшими правами"** (опционально)

### Способ 3: Через реестр

1. Нажмите **Win + R**
2. Введите `regedit`
3. Перейдите в `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
4. Создайте новый строковый параметр с именем `LayoutSwitcher`
5. Укажите путь к `LayoutSwitcher.exe`

## 🛠 Решение проблем

### Проблема: Скрипт не запускается
**Решение:**
1. Проверьте, установлен ли Python
2. Проверьте, установлены ли зависимости
3. Попробуйте запустить из командной строки с правами администратора
4. Проверьте, не блокирует ли антивирус

### Проблема: Не работает перевод текста
**Решение:**
1. Убедитесь, что текст выделен
2. Проверьте, работает ли копирование (Ctrl+C)
3. Убедитесь, что скрипт активен (иконка в трее)
4. Попробуйте перезапустить скрипт

### Проблема: Не видно иконку в трее
**Решение:**
1. Проверьте, запущен ли скрипт
2. Нажмите стрелку вверх в системном трее
3. Перезапустите скрипт

### Проблема: Ошибка импорта библиотек
**Решение:**
```bash
pip install --upgrade pyperclip pystray pillow
```

### Проблема: Скрипт не запускается в Enterprise
**Решение:**
1. Убедитесь, что вы запускаете от имени пользователя, а не администратора
2. Проверьте политики AppLocker
3. Попробуйте запустить через PowerShell: `python switcher.py`

## ❓ Часто задаваемые вопросы

### Q: Нужны ли права администратора?
**A:** Нет! Скрипт работает без прав администратора.

### Q: Будет ли антивирус блокировать скрипт?
**A:** Нет, так как скрипт не использует низкоуровневые хуки.

### Q: Можно ли запускать с флешки?
**A:** Да, соберите .exe файл и запускайте с любого носителя.

### Q: Как остановить скрипт?
**A:** Нажмите правой кнопкой мыши на иконку в трее и выберите "Exit".

### Q: Работает ли с другими раскладками?
**A:** Да, поддерживаются все раскладки Windows.

### Q: Можно ли изменить комбинацию клавиш?
**A:** Да, вы можете изменить код в файле switcher.py.
